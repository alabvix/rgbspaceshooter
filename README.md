# RGB Start Fighter

RGB Starfighter is a old school space shooter made in Game Maker 1.4.
Commands:

 - Directional keys to move the ship

 - A - Red shot
 - S - Green shot
 - D - Blue shot
 -  F - RGB shot

**Press and hold A,S or D to create a 'death energy trail'**
